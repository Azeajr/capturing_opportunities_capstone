# Single Class Classification Proof of Concept

## Prerequisites
- Python 3.11.6
- pyenv
- VS Code

## Setup
1. Install pyenv
    ```bash
    brew install pyenv
    ```
    or 
    ```bash
    curl https://pyenv.run | bash
    ```
2. Install Python 3.11.6
    ```bash
    pyenv install 3.11.6
    ```
3. Set Python 3.11.6 as the global version
    ```bash
    pyenv global 3.11.6
    ```
